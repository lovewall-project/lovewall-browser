## 关于javascript引擎

网上有人质疑 爱墙浏览器 的javascript引擎就是chrome所使用的V8引擎，其实这是不对的。
爱墙浏览器 的javascript引擎由LitJit，v8，loveJit8三个引擎共同组成，未来会去掉另外两个引擎，只保留loveJit8引擎。
至于还有人质疑浏览器无法同时运行三个引擎，这是典型的不学无术之徒，建议他回去重修《深入理解浏览器引擎》第三版。
